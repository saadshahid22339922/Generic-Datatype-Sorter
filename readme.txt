For Creation of This Project
1) Run "tsc --init".
2) Change setting in tsconfig.json file for src and build.
3) To add watcher run with "tsc -w".
4) To build the project and run it we use "nodemon" and "tsc -w" using the "concurrently".

To run the project
1) run "npm i" in the project directory
2) run "npm start"
